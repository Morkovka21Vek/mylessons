# Ping Pong game  
## Dependencies  
```sh
sudo apt install libncurses-dev catch2 # Ubuntu/Debian

sudo dnf install ncurses-devel Catch2-devel # Fedora/RHEL/CentOS

sudo pacman -S ncurses catch2 # Arch/Manjaro

brew install ncurses catch2 # macOS
```  
## Build  
```sh
make
```  
### And Run  
```sh
./build/ping_pong.out
```
