# Ping Pong game  
## dependencies
```sh
sudo apt install libncurses-dev # Ubuntu/Debian

sudo dnf install ncurses-devel # Fedora/RHEL/CentOS

sudo pacman -S ncurses # Arch/Manjaro

sudo apk add ncurses-dev # Alpine Linux

brew install ncurses # macOS
```  
## Build  
```sh
make
```  
### And Run  
```sh
./build/ping_pong.out
```
